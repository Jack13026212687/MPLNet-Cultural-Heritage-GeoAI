# Sample Output

Demo inference results are saved here by default.

Run `demo/inference.py` to generate results in this folder:

```bash
python demo/inference.py \
    --model mcpnet \
    --image sample_input/your_image.jpg \
    --output sample_output/result.png
```

Output format: side-by-side image (original | 22-class colour overlay).
